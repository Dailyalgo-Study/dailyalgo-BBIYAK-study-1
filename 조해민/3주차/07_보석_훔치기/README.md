## 3. 풀이 설명
### 💡 아이디어 도출 과정
- 첫번째부터 연속 구간에 다섯개의 보석이 있는지 확인한다.
- 나온 결과들을 비교하여 가장 수가 작은 것을 반환한다.

### ⚙️ 구현 로직의 동작 과정
1. 첫번째 배열부터 다섯 보석이 모두 포함된 구간을 확인한다.
2. 다음 배열로 넘어가서 확인을 한다.
3. 길이가 5인경우 종료한다.

### ✅ 엣지 케이스에 대한 고려 사항
- 모든 수를 확인하지 않아도 5가 나오면 종료한다.
