## 3. 풀이 설명
### 💡 아이디어 도출 과정
두 원소의 합을 구하고 target과 같은 경우 return 한다.
그 중 차가 가장 작은 값을 반환
다른 경우에는 -1. -1을 return한다.

### ⚙️ 구현 로직의 동작 과정
1. 처음의 원소와 마지막 원소의 합을 구한다.
2. 합이 target보다 작으면 두번째 원소와 마지막 원소의 합을 구한다.
3. 합이 target보다 크면 첫번재 원소와 뒤에서 두번째 원소의 합을 구한다.
4. 합이 같은 경우 원소를 반환한다.
5. 그 중 두 원소의 차가 가장 작은 것을 반환한다.
6. 합이 target과 같은것이 없다면 -1, -1을 반환한다.

### ✅ 엣지 케이스에 대한 고려 사항
- 두 원소의 차가 가장 작은 쌍 반환하는 코드가 필요하다
