### 💡 아이디어 도출 과정
- 한 자리부터 전체 문자열까지의 문자열을 만든다.
- 각각의 문자열이 펠린드롭인지 확인한다.

### ⚙️ 구현 로직의 동작 과정
1. text를 사용하여 만들 수 있는 문자열을 확인한다.
2. 만든 문자열의 좌우를 비교하여 같은지 다른지 확인한다.
3. 같다면 길이를 구하고 최대 길이를 반환한다.
