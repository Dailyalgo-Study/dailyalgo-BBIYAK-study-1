## 3. 풀이 설명
### 💡 아이디어 도출 과정
- 각 문자를 중심으로 좌우를 비교한다.
- 범위를 벗어날때까지 비교한다.

### ⚙️ 구현 로직의 동작 과정
1. 하나의 문자를 중심으로 좌우를 비교한다.
2. 길이를 하나씩 늘려가면서 펠린드롭인지 확인하고 그 길이를 확인한다.
3. 가장 긴 길이의 값을 반환한다.

### ✅ 엣지 케이스에 대한 고려 사항
- 2번 문제의 코드를 사용한 경우 time out이 발생한다.

## 4. 어려웠던 점 및 해결 과정
