## 3. 풀이 설명
### 💡 아이디어 도출 과정
배열에서 두 원소의 합이 target이 되는 개수를 센다.

### ⚙️ 구현 로직의 동작 과정
1. 배열에서 두 원소의 합을 구한다.
2. 배열의 모든 원소의 값이 같은 경우 모든 경우의 수를 구한다.
3. 합이 target보다 작은 경우 두번째 배열로 이동한다.
4. 합이 target보다 큰 경우 마지막 배열을 왼쪽으로 이동한다.

### ✅ 엣지 케이스에 대한 고려 사항

## 4. 어려웠던 점 및 해결 과정
원소가 모두 동일한 경우에는 모든 조합을 구해야 한다.
