## 3. 풀이 설명
### 💡 아이디어 도출 과정
1. 배열을 오름차순한다.
2. 가장 가벼운 사람과 무거운 사람을 태운다.
3. 둘의 합이 limit보다 크면 무거운 사람만 태운다.
4. 작으면 둘 다 태우고 보트의 수를 1 증가시킨다.

### ⚙️ 구현 로직의 동작 과정
1. 사람을 오름차순 정렬한다.
2. 맨 첫번째 배열과 맨 마지막 배열의 합이 limit보다 큰지 작은지 확인한다.
3. 합이 작으면 수를 센다.
4. 합이 크면 한 사람만 태운다.
