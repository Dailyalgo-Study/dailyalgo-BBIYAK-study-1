### ⚙️ 구현 로직의 동작 과정
1. 배열을 문자열로 변환한다.
2. 문자열을 이어붙였을 때 더 큰 수가 어떤것인지 비교한다.
3. 정렬된 문자열을 이어붙인다.
4. 0으로 시작하는 경우 0을 반환한다.
