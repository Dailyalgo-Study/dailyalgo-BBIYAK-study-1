## 4. 어려웠던 점 및 해결 과정
N = 100000이라 시간 복잡도인 O(NlogN) 안에 해당될까 걱정 됨
강의에서 log1000000 = 20, logN은 생각보다 굉장히 작다. (20-50 정도라고 생각하면 됨) 라고 노트해놓은게 있어서 괜찮다고 생각했음.
