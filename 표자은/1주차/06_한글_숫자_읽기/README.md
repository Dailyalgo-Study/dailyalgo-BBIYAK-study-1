## 4. 어려웠던 점 및 해결 과정

한글이라서 replace 쓰는게 어려움. replace 파라미터는 계속 까먹는다. size_t도 까먹고. char -> string 변환도 까먹고. 지피티 돌렸다.


## 5. 궁금한 점

한글은 char로 하면 안되는지. 한글 문자의 크기는 어떻게 되는지, 알파벳과 뭐가 다른지. 답안 몇개 테케에서 틀렸다고 하는데 왜인지
