## 3. 풀이 설명

### 💡 아이디어 도출 과정

>
> 윈도우의 구간을 한칸씩 옮기며 각 구간의 합을 구합니다.
>
> 첫 윈도우 합을 구하고 이후 첫 윈도우 인덱스 값을 빼고 옆에있는 새 윈도우 인덱스를 더해가는 방식으로 모든 윈도우 구간의 합을 구합니다.

### ⚙️ 구현 로직의 동작 과정

>
> 1. for문으로 첫 윈도우의 합을 구합니다.
> 2. 다른 for문으로 첫 윈도우 부터 윈도우 첫번째 인덱스 값(numbers[i]을 빼고 새로 들어올 윈도우 인덱스 값(numbers[i+window])을 더한값을 answer에 추가합니다.
> 3. answer를 반환합니다.
