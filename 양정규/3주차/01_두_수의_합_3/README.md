# 01. 두 수의 합 3

## 1. 문제 정보

- [데일리알고 - 두 수의 합 3](https://dailyalgo.kr/ko/problems/194)

## 2. 소요 시간

- 60분

## 3. 풀이 설명

### 💡 아이디어 도출 과정

>
> 배열의 두 원소의 차가 가장 작으며 두 원소의 target값과 같은 두 원소를 구합니다.
>
> 투 포인터를 이용해 양 끝에서 가운데로 모이는 방향으로 두 원소의 관계를 차례대로 구합니다.

### ⚙️ 구현 로직의 동작 과정

>
> 1. while문을통해 왼쪽원소값(numbers[left])과 오른쪽원소값(numbers[right])의 합을 target과 비교를 통해 왼쪽 또는 오른쪽 원소값을 옆으로 이동합니다.
> 2. 처음으로 두 원소의 합이 target과 같다면 두 원소를 answer에 입력합니다.
> 3. 다른 원소 합이 target과 같고 두 원소의 차와 answer에 두 원소의 차와 비교하여 더 작은값을 answer에 갱신합니다.
> 4. 왼쪽 인덱스를 오른쪽으로 한칸 이동합니다.
> 5. 두 인덱스의 차가 1이 되면 while문을 종료합니다.
> 6. answer를 반환합니다.
