# 05. 배열 병합하기

## 1. 문제 정보

- [데일리알고 - 배열 병합하기](https://dailyalgo.kr/ko/problems/195)

## 2. 소요 시간

- 60분

## 3. 풀이 설명

### 💡 아이디어 도출 과정

>
> 배열안의 두 원소의 합이 target값과 같으며 두 값의 차이가 가장 작은 원소 쌍을 구한다.
>
> 투포인터를 이용해 배열 양끝에서 시작해서 가운데로 좁혀오며 답을 구한다.

### ⚙️ 구현 로직의 동작 과정

>
> 1. 양 끝의 원소부터 선택하여 두 원소의 합을 target과 비교하여 조건에 따라 옆의 인덱스로 이동한다.
> 2. 두 원소의 합이 target과 같으며 answer가 초기값(-1,-1)이라면 answer를 두 원소로 갱신한다.
> 3. 또 다른 두 원소의 합이 target과 같으며 두 원소의 차(diff)가 answer의 두 원소의 차보다 작다면 answer에 새로운 두 원소로 갱신한다.
> 4. 왼쪽 인덱스를 오른쪽으로 한 칸 이동한다.
> 5. 두 원소의 합이 target과 같지않으며 두 인덱스의 차가 1이면 while문을 종료한다.
> 6. answer를 반환한다.
