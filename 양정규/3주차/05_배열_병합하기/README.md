# 05. 배열 병합하기

## 1. 문제 정보

- [데일리알고 - 배열 병합하기](https://dailyalgo.kr/ko/problems/206)

## 2. 소요 시간

- 30분

## 3. 풀이 설명

### 💡 아이디어 도출 과정

>
> 두 배열을 하나로 합쳐 오름차순 정렬합니다.
>
> 두 배열을 첫번째 원소부터 크기비교를 하여 순서에 맞게 새로운 빈 배열에 입력합니다.

### ⚙️ 구현 로직의 동작 과정

>
> 1. 첫 while문으로 arr1의 첫 인덱스(arr1[arr1Index])와 arr2(arr2[arr2Index])의 첫 인덱스 비교부터 시작하여 작은 값을 빈 배열(answer)에 추가합니다.
> 2. 두,세번째 while문으로 첫 while문으로 추가되지 못한 남은 배열의 인덱스 값을 차례대로 answer에 추가합니다.(두 배열의 길이가 다르면 하나의 배열에 추가되지 못한 남은 원소값이 있습니다)
> 3. answer를 반환합니다.
