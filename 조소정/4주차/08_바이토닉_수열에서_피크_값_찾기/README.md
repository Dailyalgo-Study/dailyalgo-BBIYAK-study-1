## 3. 풀이 설명
### 💡 아이디어 도출 과정

이진탐색을 해서 오름에서 내림차순이 되는 순간을 찾습니다.

### ⚙️ 구현 로직의 동작 과정

1. 중앙 값을 계산합니다.
2. mid 와 mid +1 을 비교합니다.
3. 크다면 left 를 증가
4. 작다면 right 를 감소 시킵니다.
5. left == right 과 된다면 그 값이 피크이기때문에 반환합니다.
