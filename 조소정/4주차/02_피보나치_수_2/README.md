# 07. 피보나치 수 2

## 1. 문제 정보
- [데일리알고 - 피보나치 수 2](https://dailyalgo.kr/ko/problems/209)

## 2. 소요 시간
- 60분

## 3. 풀이 설명
### 💡 아이디어 도출 과정

피보나치를 계산하는 과정에서 재귀함수가 몇번 호출되는지
도출하는 문제라서 재귀 함수를 일단 만들고
그걸 호출할 때마다 카운트해주는 변수를 둬서 답을 찾으면 될것같습니다.

### ⚙️ 구현 로직의 동작 과정

1. n ==1 일 때와 n==2일 때 호출되는 횟수를 체크합니다
2. 답을 반환합니다.

## 4. 어려웠던 점 및 해결 과정
