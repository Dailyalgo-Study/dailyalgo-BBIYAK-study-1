### 💡 아이디어 도출 과정

이진탐색을 이용해서 타겟을 찾습니다.

### ⚙️ 구현 로직의 동작 과정

1. 최소와 최대 범위를 지정합니다.
2. 중앙 값을 구한 후 타겟과 비교합니다
3. 작으면 최소 범위를 증가시키고
4. 크면 최대 범위를 줄입니다.
5. 모든 배열을 탐색했는데 없으면 false 를 반환합니다.
