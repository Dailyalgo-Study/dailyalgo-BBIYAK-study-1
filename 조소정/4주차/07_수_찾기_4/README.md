# 07. 수 찾기 4

## 1. 문제 정보

## 2. 소요 시간

## 3. 풀이 설명
### 💡 아이디어 도출 과정

이진탐색으로 범위를 좁혀가면서 선택된 인덱스의 값을 타겟과 비교합니다. 같다면 바로 반환하고 작거나 크다면 그 차이 값과 비교 값을 기록한 다음 탐색이 끝날때까지 비교 후 가장 큰 값을 반환합니다.

### ⚙️ 구현 로직의 동작 과정

1. 중앙 값을 계산하여 타겟과 비교합니다
2. 같은 값이면 반환합니다
3. 차이를 비교했을 때 같다면 그 중 더 큰 수를 선택합니다.



### ✅ 엣지 케이스에 대한 고려 사항

## 4. 어려웠던 점 및 해결 과정

처음에 gap을 0으로 별생각없이 초기화해서 답이 안나왔습니다.
그리고 gap을 매우 큰 값으로 바꿔주니 제대로 동작했습니다.
