## 3. 풀이 설명
### 💡 아이디어 도출 과정
팰린드롬 문자열은 앞뒤가 똑같은 문자열 을 뜻하니까 
모든 부분 문자열을 만들어보고 그게 팰린드롬인지 하나씩 
확인하는 방법을 생각했습니다.

### ⚙️ 구현 로직의 동작 과정

1. 양끝의 인덱스를 구합니다.
2. 각 범위부터 한칸씩 이동하며 문자가 같은지 확인합니다.
3. 맞다면 길이를 계산해서 반환합니다.
4. 아니라면 인덱스를 이동해 다시 계산합니다.
