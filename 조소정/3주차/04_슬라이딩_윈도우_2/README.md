## 2. 소요 시간

## 3. 풀이 설명
### 💡 아이디어 도출 과정

강의시간에 배웠던게 기억이 나서 
한 칸을 옮길 때 마다 이전 합에서 왼쪽값을 빼고, 새로 들어온 오른쪽 값을
더하는 방식으로 처리했습니다.

### ⚙️ 구현 로직의 동작 과정
1. 첫번째 윈도우의 합을 구합니다.
2. 이후 한 칸씩 오른쪽으로 이동하면서 왼쪽의 값을 빼고 새로운 값을 더합니다.
3. 합을 백터에 저장합니다.
