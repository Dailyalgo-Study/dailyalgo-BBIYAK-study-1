# 07. 보석 훔치기

## 1. 문제 정보
- [데일리알고 - 보석 훔치기](https://dailyalgo.kr/ko/problems/205?language=cpp)

## 2. 소요 시간
- 90분

## 3. 풀이 설명
### 💡 아이디어 도출 과정

슬라이딩 방법을 활용해서 각 구간을 탐색합니다.
map 에 보석을 모두 저장하여 1종류씩 들어와있는지 체크합니다.
조건을 만족하면 답을 반환합니다.



### ⚙️ 구현 로직의 동작 과정

1. map<string, int>으로 구간 안의 보석 개수를 셉니다.
2. right 포인터를 한 칸씩 이동시키면서 보석을 추가합니다.
3. 구간 안에 보석이 5종류 모두 들어왔는지 확인합니다.
4. 5종류가 모두 있다면 left를 오른쪽으로 옮기며 구간을 최대한 줄입니다.
5.줄이는 동안 최소 구간 길이를 계속 갱신합니다.
6.모든 과정을 마치면 minLen에 가장 짧은 구간 길이가 저장됩니다.
