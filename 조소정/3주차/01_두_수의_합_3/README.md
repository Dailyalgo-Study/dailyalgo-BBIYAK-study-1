# 01. 두 수의 합 3

## 1. 문제 정보
- [데일리알고 - 두 수의 합 3](https://dailyalgo.kr/ko/problems/194)

## 2. 소요 시간
- 60분

## 3. 풀이 설명
### 💡 아이디어 도출 과정
양쪽 끝에서부터 값을 하나씩 선택해서 target의 값과 비교합니다.
뽑은 값이 같은 경우엔 오름차순 정렬이기때문에
모든 원소의 값이 동일하다고 생각하고 답을 처리합니다.



### ⚙️ 구현 로직의 동작 과정

1. 두 값의 합이 target보다 작을 때 front의 인덱스를 올립니다.
2. 두 값의 합이 target 과 같고 각각의 값이 다를 때는 두 인덱스 모두 바꿉니다.
3. 두 값의 합이 target과 같고 각각의 값이 같을 때는 전체 원소의 개수를 구하고
   각 쌍의 개수를 구해 답을 반환합니다.
4. 두 값의 합이 target보다 클 때 end 의 인덱스를 감소합니다.
