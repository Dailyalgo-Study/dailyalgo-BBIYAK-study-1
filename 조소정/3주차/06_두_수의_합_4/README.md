# 06. 두 수의 합

## 1. 문제 정보
- [데일리알고 - 두 수의 합](https://dailyalgo.kr/ko/problems/195)

## 2. 소요 시간
- 60분

## 3. 풀이 설명
### 💡 아이디어 도출 과정

투포인터를 이용해서 target 과 같은 값이 되는 쌍을 찾고
각 쌍의 차이를 저장한 뒤 가장 작은 차이가 나는 값을 반환합니다.



### ⚙️ 구현 로직의 동작 과정

1. 양 끝 값의 합을 구합니다.
2. 작으면 left 인덱스를 증가시킵니다.
3. 크면 right 인덱스를 감소시킵니다.
4. target 과 같은 값이라면 차이를 구합니다.
5. 기존의 차이 값과 비교하여 더 작은 쌍을 저장합니다.
