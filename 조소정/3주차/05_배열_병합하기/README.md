### 💡 아이디어 도출 과정

첫번째 배열의 뒤에 2번째 배열을 붙입니다.
정렬하여 답을 반환합니다.


### ⚙️ 구현 로직의 동작 과정

1. 첫번째 배열을 넣습니다.
2. 두번째 배열을 넣습니다.
3. 배열을 정렬하여 답을 반환합니다.
