## 3. 풀이 설명
### 💡 아이디어 도출 과정

가장 큰 값끼리 곱해야 최대값이 나올 수 있다는 점을 이용했습니다.
그래서 배열을 오름차순하여 각 인덱스의 값을 곱했습니다.


### ⚙️ 구현 로직의 동작 과정

1. 배열을 오름차순합니다.
2. 각 인덱스의 값을 곱하여 더합니다.
