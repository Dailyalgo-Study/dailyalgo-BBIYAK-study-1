## 3. 풀이 설명
### 💡 아이디어 도출 과정
> 문제를 어떻게 이해했고, 어떤 접근 방법을 떠올렸는지 설명합니다.

숫자와 한글을 매핑시켜놓고 순회하면서 맞는 단어가 있으면 대체하는 방법입니다.
일곱과 일 은 문자열의 첫번째가 겹치기때문에 길이를 먼저 비교하고 일치하는지를 확인해야합니다.

그래서 이건 제가 아직 풀기 힘들다고 생각하고 다시 제대로 이해가 되면 풀어보고 싶어서 넘어갔습니다.
