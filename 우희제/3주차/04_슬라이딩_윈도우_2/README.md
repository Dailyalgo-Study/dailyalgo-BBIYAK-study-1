# 04. 슬라이딩 윈도우 2

## 1. 문제 정보
- [데일리알고 - 슬라이딩 윈도우 2](https://dailyalgo.kr/ko/problems/203)

## 2. 소요 시간
- 1시간

## 3. 풀이 설명
### 💡 아이디어 도출 과정

1. 첫번째 윈도우의 합을 먼저 구한다.
2. 두번째 구간부터 이전 합을 이용해 이전합 - 빠진값 + 새 값으로 구한다.

### ⚙️ 구현 로직의 동작 과정

1. 주어진 window 수 만큼 반복하면서 첫번째 윈도우의 합을 구한다.
2. 첫번째 윈도우의 합 sum을 answer에 추가한다.
3. left를 1부터 해서 현재 윈도우의 시작 인덱스로 하고 size - window까지 반복한다.
4. 현재 윈도우의 끝 인덱스(right)를 구하고 sum에 새로 들어온 numbers[right]를 더한 다음 이전 윈도우에서 나가는 numbers[left - 1]를 빼서 sum을 갱신한다.
5. 갱신한 sum을 answer에 추가한다.
