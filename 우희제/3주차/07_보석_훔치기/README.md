## 3. 풀이 설명
### 💡 아이디어 도출 과정

슬라이딩 윈도우 방식으로 문제 풀기!

1. right 포인터로 jewels 사이즈만큼 돌면서 보석 종류 세기
2. left 포인터로 구간을 줄이면서 5가지 보석을 포함한 제일 짧은 구간을 찾는다.
	min() 함수 이용해서 길이를 비교하고 저장하기

### ⚙️ 구현 로직의 동작 과정

1. 보석 이름을 인덱스로 만들어 접근할 수 있게 jewelindex() 함수를 만든다.
2. jewelcount(5) 보석 개수만큼 배열을 만들어 여기에 현재 구간에 있는 보석의 개수를 저장하고 count 변수로 보석 종류를 관리한다.
3. 첫번째 인덱스부터 오른쪽으로 포인터를 이동 시키면서 새로운 보석이 등장하면 count를 증가 시키고 jewelcount()에 해당 보석 개수를 증가 시킨다.
4. count가 5일 경우 반복문을 돌면서 min(answer, right - left + 1) 최소 길이의 구간을 찾아 answer에 저장한다.
	left 포인터로 왼쪽 보석을 제거하고 만약 현재 보석의 개수가 0이 되면 보석이 없다는 뜻으로 count를 감소 시킨다.
