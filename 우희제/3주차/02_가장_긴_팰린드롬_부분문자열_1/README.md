### 💡 아이디어 도출 과정

1. 문자열의 시작 인덱스와 끝 인덱스를 찾아 팰린드롬인지 확인한다.
2. 맞다면 길이의 최대값을 구한다.

### ⚙️ 구현 로직의 동작 과정

1. i는 0부터 j는 i+1부터 이중for문을 돌면서 시작문자와 끝문자를 비교하여 팰린드롬인지 검사한다.
2. 다르면 break로 멈추고 팰린드롬이 맞다면 길이를 계산한다. len = j - i + 1
3. len가 answer보다 크가 새로 answer에 저장해 최대 길이의 팰린드롬을 구한다.
