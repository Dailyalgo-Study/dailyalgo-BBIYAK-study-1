# **팩토리얼**

## **1. 문제 정보**

- [데일리알고 - 팩토리얼](https://dailyalgo.kr/ko/problems/207)

## **2. 소요 시간**

- 5분

## **3. 풀이 설명**

### **💡 아이디어 도출 과정**

- 팩토리얼은 n! = n × (n-1)! 로 표현할 수 있으며,
    
    n이 1이 될 때까지 계속 자기 자신을 호출하는 재귀 함수로 표현할 수 있다.
    
- 1! = 1 이라는 종료 조건을 명시하면 된다.
- 문제의 입력 범위(1 ≤ n ≤ 20)에서는 long 자료형으로 충분히 표현 가능하다.

### **⚙️ 구현 로직의 동작 과정**

1. 자연수 n을 입력받는다.
2. factorial() 함수를 재귀적으로 호출한다.
3. n이 1이면 1을 반환하고, 아니라면 n × factorial(n-1)을 반환한다.
4. 결과를 출력한다.

### **✅ 엣지 케이스에 대한 고려 사항**

## **4. 어려웠던 점 및 해결 과정**

- 반복문에 비해 재귀는 종료 조건을 명확히 하지 않으면 StackOverflowError가 발생한다.
- 따라서 n == 1일 때 반드시 반환해야 한다. (Base Case)

## **5. 궁금한 점**

- 없습니다.

## **전체 코드**

```java
import java.util.*;

public class Solution {
	public long solution(int n) {
		if (n == 1) return 1;
		return n * solution(n - 1);
	}
}
```
