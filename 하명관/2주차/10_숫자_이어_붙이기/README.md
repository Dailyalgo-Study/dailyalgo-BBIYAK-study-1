# 10. 숫자 이어 붙이기

## 1. 문제 정보

- [데일리알고 - 숫자 이어 붙이기](https://dailyalgo.kr/ko/problems/192)

## 2. 소요 시간

- 30분

## 3. 풀이 설명

### 💡 아이디어 도출 과정

- a + b 와 b + a 중 더 큰 쪽이 앞에 오도록 정렬하면 된다.
- 예를 들어 [3, 30, 34]의 경우 숫자 크기 기준으로 하면 34 30 3 → 34303이 되지만 3 34 30 순서로 정렬하면 33430이 더 큰 수가 된다.

### ⚙️ 구현 로직의 동작 과정

- 숫자 배열을 문자열 배열로 변환한다.
    
    ```java
    List<String> list = new ArrayList<>();
    
    for (int num : numbers) {
        list.add(String.valueOf(num));
    }
    
    String[] arr = list.toArray(new String[0]);
    ```
    
- 문자열을 이어붙였을 때 더 큰 조합이 앞에 오도록 정렬한다.
    - `Arrays.sort(arr, (a, b) -> (b + a).compareTo(a + b));`
- 정렬된 문자열들을 순서대로 이어 붙인다.
    - `String result = String.join("", arr);`
- 모든 문자가 0인경우 0만 반환
    - `if (result.charAt(0) == '0') return "0";`

### ✅ 엣지 케이스에 대한 고려 사항

- 모든 숫자가 0일 경우 0만 반환하도록 해야 한다
- 숫자를 모두 문자열로 비교해야 한다.

## 4. 어려웠던 점 및 해결 과정

- 3과 30을 비교할 때 단순히 숫자 크기가 아니라 문자열 형태로 만들었때의 크기를 비교해야하는 점이 어려웠음.
