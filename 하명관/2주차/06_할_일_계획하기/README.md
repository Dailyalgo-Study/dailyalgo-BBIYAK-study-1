# 06. 할 일 계획하기

## 1. 문제 정보

- [데일리알고 - 할 일 계획하기](https://dailyalgo.kr/ko/problems/186)

## 2. 소요 시간

- 20분

## 3. 풀이 설명

### 💡 아이디어 도출 과정

- 시작 시간, 끝 시간이 주어질 때 수행할 수 있는 최대 작업 개수를 반환해야 한다.
- 만약 진행죽인 작업이 있는 경우 다른 작업을 시작 할 수 없다.

### ⚙️ 구현 로직의 동작 과정

- 끝 시간을 기준으로 오름차순 정렬을 한다.
    
    ```java
            Arrays.sort(tasks, (a, b) -> {
                return a[1] - b[1];
            });
    ```
    
- 시작 시간이 이전 작업이 끝나는 시간보다 크거나 같다면 작업이 가능한 경우이다.
다만 시작 시간이 변경되면 끝나는 시간도 새로 업데이트 해줘야 한다.
    
    ```java
        int lastEnd = -1;
    		int answer = 0;
    
            for (int[] t : tasks) {
                int start = t[0];
                int end = t[1];
                
                if (start >= lastEnd) {
                    lastEnd = end;
                    answer++;
                }
            }
    ```
