# 04. 슬라이딩 윈도우 2

## 1. 문제 정보

- [데일리알고 - 슬라이딩 윈도우 2](https://dailyalgo.kr/ko/problems/203)

## 2. 소요 시간

- 15분

## 3. 풀이 설명

### 💡 아이디어 도출 과정

- 슬라이딩 윈도우를 사용해서 풀면 된다.
- 단순하게 각 구간마다 합을 다시 계산하면 O(N × window)로 비효율적이다.
- 이전 구간의 합을 활용하면, 한 칸 이동할 때마다 맨 앞 원소를 빼고 맨 뒤 원소를 더하기만 하면 된다.

### ⚙️ 구현 로직의 동작 과정

1. 첫 번째 윈도우(0~window-1)의 합을 구한다.
    
    ```java
            for (int i = 0; i < window; i++) {
                sum += numbers[i];
            }
    ```
    
2. 그 합을 결과 리스트에 추가한다.
    - `result.add(sum);`
3. 이후부터는 한 칸씩 오른쪽으로 이동하면서, 이전 윈도우의 맨 앞 원소를 빼고, 새로 들어온 원소를 더해 합을 업데이트한다.
    
    ```java
           for (int i = window; i < n; i++) {
                sum += numbers[i] - numbers[i - window];
                result.add(sum);
            }
    ```
    
4. 각 단계에서 계산된 합을 결과 리스트에 추가한다.
    
    ```java
    return result.stream().mapToInt(Integer::intValue).toArray();
    ```
