### ⚙️ 구현 로직의 동작 과정
- numbers 길의 변수 초기화
- (numbers의 길이) - (window 크기) + 1
  - range(n,m)은 m값을 포함시키지 않기 때문에 +1을 해준다. n이상 m미만
- answer에 추가할 pre_answer초기화
- 다시 for문을 돌면서 각 구간의 값을 더한다.
- answer에 넣는다.
