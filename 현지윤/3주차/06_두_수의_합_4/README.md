## 3. 풀이 설명
### 💡 아이디어 도출 과정
target이 되는 경우에 수를 모두 찾고 절댓값이 작은 경우를 리턴한다.

### ⚙️ 구현 로직의 동작 과정
- 양끝에서 가운데로 이동하도록
- 2개의 원소의 합이 타겟값과 같다면 두 수의 차를 비교한다.
- 오름차순이므로 right - left로 
- 만약 두 수의 차가 더 작다면 min값을 바꾼다.
- answer에 담아 반환
- 값이 타겟보다 작다면 왼쪽 인덱스 증가
- 값이 타겟보다 크다면 오른쪽 인덱스 감소
