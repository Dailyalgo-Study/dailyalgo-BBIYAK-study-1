## 3. 풀이 설명
### 💡 아이디어 도출 과정
- 선택 정렬은 같은 값에 대해서 위치가 깨질 수 있으므로 버블정렬로구현
- 0번째 인덱스 부터 n-1번째 인덱스까지 두 원소를 비교해 자리를 교체한다.

### ⚙️ 구현 로직의 동작 과정
- 가까운 두 원소를 비교하여 큰 값을 앞으로 보낸다.
- 한 번 순회를 돌면 가장 작은 값이 맨 뒤로 이동이 된다.
- 이를 반복하여 전체를 정렬한다.
