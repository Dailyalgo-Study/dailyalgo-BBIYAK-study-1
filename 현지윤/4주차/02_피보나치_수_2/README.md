### ⚙️ 구현 로직의 동작 과정
fibo(5) 일 경우, 재귀함수 동작과정
```
fibo(5)
  -> fibo(4) + fibo(3) -> 종료조건을 만족하지 못함
       fibo(4) => fibo(3) + fibo(2)
       fibo(3) => fibo(2) + fibo(1)
       fibo(2) => fibo(1) + fibo(0) => 1
즉, fibo(5) 
    fibo(5)
    ==> fibo(4) + fibo(3)
    ==> (fibo(3)+fibo(2)) + (fibo(2)+ fibo(1))
    ==> (fibo(2)+fibo(1)+ fibo(2)) + (1+0)
    ==> (1+1+1) + (1+1)  ==> 3 + 2 => 5



```
